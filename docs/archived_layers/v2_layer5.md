## COMMIT 3: Solution Provenance & Reproducibility System

### The problem this solves

Professor finishes top 3. The hosts say "show us your code." Professor can't answer — the code was generated by LLMs, executed in subprocesses, and discarded after output was captured. Without provenance, Professor cannot claim prizes, cannot post writeups, and cannot prove results aren't fabricated.

### Three components, tightly coupled

**Component 1** — Code Ledger (captures everything)
**Component 2** — Solution Assembler (stitches into notebook)
**Component 3** — Writeup Generator (narrative from captured reasoning)

### Component 1: Code Ledger (tools/code_ledger.py)

The Code Ledger already exists from Layer 0 inside `run_in_sandbox()` — it captures every sandbox execution as a JSONL entry. This commit UPGRADES it with the full entry schema and a query interface.

**Full entry schema — every field must be captured per sandbox execution:**

```python
@dataclass
class CodeLedgerEntry:
    # Identity
    entry_id: str          # Sequential: "001", "002", etc.
    timestamp: str         # ISO 8601: "2026-04-15T08:30:00Z"
    
    # Source
    agent: str             # Which agent generated this code: "feature_factory", "ml_optimizer", etc.
    purpose: str           # Human-readable: "Round 2: frequency encoding for category columns"
    round_num: int         # For Feature Factory: which round (1-5). 0 for non-round agents.
    attempt: int           # Retry attempt number (1-4 from Self-Debugging cascade)
    
    # Code
    code: str              # EXACT Python code that was executed
    code_hash: str         # SHA-256 hash of code, first 16 chars
    
    # Context
    inputs: list           # File paths read by this code: ["clean_train.parquet"]
    outputs: list          # File paths written: ["features_train.parquet"]
    dependencies: list     # Entry IDs this code depends on: ["001", "003"]
    
    # Result
    success: bool          # Did execution succeed?
    stdout: str            # Truncated to 2000 chars
    stderr: str            # Truncated to 2000 chars
    runtime_seconds: float # Wall-clock execution time
    
    # Decision context
    llm_prompt: str        # The prompt that generated this code, truncated to 3000 chars
    llm_reasoning: str     # LLM's explanation/chain-of-thought, truncated to 1000 chars
    
    # Survival
    kept: bool             # Did this code's output survive the pipeline? Default: True
    rejection_reason: str  # If kept=False: why it was rejected
```

**Integration — how agents populate these fields:**

Each agent already calls `run_in_sandbox()` with `agent_name`, `purpose`, `round_num`, `attempt`, `llm_prompt`, `llm_reasoning`. The sandbox function populates the rest automatically:
- `entry_id`: sequential counter maintained by the ledger module
- `timestamp`: captured at execution time
- `code`: the code string passed to sandbox
- `code_hash`: `hashlib.sha256(code.encode()).hexdigest()[:16]`
- `inputs`/`outputs`: parsed from the code (detect `read_csv`, `read_parquet`, `write_csv`, `write_parquet` calls and extract paths)
- `dependencies`: inferred from input/output overlap with previous entries (if entry 005 reads a file that entry 003 wrote, 005 depends on 003)
- `success`, `stdout`, `stderr`, `runtime_seconds`: from subprocess result
- `kept`: True by default
- `rejection_reason`: empty by default

**Rejection marking — called by gates and Critic:**

```python
def mark_rejected(entry_id: str, reason: str, session_dir: str) -> None:
    """
    Mark a Code Ledger entry as rejected (kept=False).
    
    Called by:
    - Null importance gate: reason="Null importance: p=0.3421 > 0.05"
    - Wilcoxon gate: reason="Wilcoxon: p=0.0823 > 0.05"
    - Critic rejection: reason="Critic CONFIRMED_CRITICAL: preprocessing leakage"
    - Retry supersedes: reason="Superseded by attempt 3"
    """
    entries = _read_ledger(session_dir)
    for entry in entries:
        if entry["entry_id"] == entry_id:
            entry["kept"] = False
            entry["rejection_reason"] = reason
            break
    _write_ledger(entries, session_dir)
```

**Query interface — for Solution Assembler and Writeup Generator:**

```python
def get_kept_entries(session_dir: str) -> list[dict]:
    """Return all entries with kept=True, ordered by entry_id."""
    entries = _read_ledger(session_dir)
    return sorted(
        [e for e in entries if e.get("kept", True)],
        key=lambda e: int(e["entry_id"])
    )

def get_entries_by_agent(session_dir: str, agent_name: str) -> list[dict]:
    """Return all entries (kept and rejected) for a specific agent."""
    entries = _read_ledger(session_dir)
    return [e for e in entries if e["agent"] == agent_name]

def get_reasoning_chain(session_dir: str) -> list[dict]:
    """
    Return kept entries with reasoning fields for writeup generation.
    Strips code and prompts to reduce size — keeps only the narrative-relevant fields.
    """
    kept = get_kept_entries(session_dir)
    return [
        {
            "agent": e["agent"],
            "purpose": e["purpose"],
            "reasoning": e.get("llm_reasoning", "No reasoning provided."),
            "round": e.get("round_num", 0),
            "success": e["success"],
            "rejection_reason": e.get("rejection_reason", ""),
        }
        for e in kept
    ]

def get_rejected_entries(session_dir: str) -> list[dict]:
    """Return all rejected entries with rejection reasons — for 'What Didn't Work' section."""
    entries = _read_ledger(session_dir)
    return [e for e in entries if not e.get("kept", True)]
```

**Storage format:** JSONL (one JSON object per line). Typical competition: 30-55 entries, ~150KB code + metadata, ~500KB with truncated prompts. Negligible storage.

```python
def _read_ledger(session_dir: str) -> list[dict]:
    path = os.path.join(session_dir, "code_ledger.jsonl")
    if not os.path.exists(path):
        return []
    with open(path, "r") as f:
        return [json.loads(line) for line in f if line.strip()]

def _write_ledger(entries: list[dict], session_dir: str) -> None:
    path = os.path.join(session_dir, "code_ledger.jsonl")
    with open(path, "w") as f:
        for entry in entries:
            f.write(json.dumps(entry) + "\n")

def _append_entry(entry: dict, session_dir: str) -> None:
    path = os.path.join(session_dir, "code_ledger.jsonl")
    with open(path, "a") as f:
        f.write(json.dumps(entry) + "\n")
```

### Component 2: Solution Assembler (tools/solution_assembler.py)

**The hard problem:** Code block 003 from Feature Factory uses `df`. Code block 001 from Data Engineer also uses `df`. When stitched together, which `df`? The sandbox runs each block in isolation, but the notebook needs them to flow linearly.

**Resolution strategy:**
1. Topological sort by agent pipeline position → round → entry_id
2. Group into stages: imports → data_loading → feature_engineering → model_training → ensemble → postprocessing → submission
3. Extract and deduplicate ALL import statements across all blocks
4. Clean sandbox artifacts: remove SANDBOX_PREAMBLE duplicates, replace sandbox file paths with relative paths, pin random seeds explicitly
5. Add stage headers with comments explaining each block's purpose

```python
AGENT_ORDER = [
    "data_engineer",
    "feature_factory",
    "creative_hypothesis",
    "ml_optimizer",
    "ensemble_architect",
    "post_processor",
]

def assemble_solution_notebook(state: ProfessorState, session_dir: str) -> str:
    """
    Stitch all kept Code Ledger entries into a single reproducible Python script.
    
    The script has:
    - ZERO Professor dependencies (no imports from professor/)
    - ZERO LLM calls, ZERO API dependencies
    - Only standard ML libraries (polars, numpy, scipy, sklearn, lightgbm, xgboost, catboost)
    - Deterministic (all random seeds pinned)
    - Can be run by anyone with Python 3.11 + requirements.txt
    
    Returns: path to solution_notebook.py
    """
    kept = get_kept_entries(session_dir)
    
    # 1. Group by agent in pipeline order
    grouped = {}
    for agent in AGENT_ORDER:
        agent_entries = sorted(
            [e for e in kept if e["agent"] == agent and e["success"]],
            key=lambda e: (e.get("round_num", 0), int(e["entry_id"]))
        )
        if agent_entries:
            grouped[agent] = agent_entries
    
    # 2. Extract and deduplicate imports
    all_imports = set()
    for entries in grouped.values():
        for entry in entries:
            for line in entry["code"].split("\n"):
                stripped = line.strip()
                if stripped.startswith("import ") or stripped.startswith("from "):
                    # Skip Professor-internal imports
                    if any(pkg in stripped for pkg in ["professor", "tools.", "agents.", "shields.", "graph."]):
                        continue
                    all_imports.add(stripped)
    
    # 3. Build the notebook
    notebook_lines = [
        "#!/usr/bin/env python3",
        '"""',
        f"Solution Notebook — {state.competition_name}",
        f"Generated by Professor v2 on {datetime.utcnow().strftime('%Y-%m-%d %H:%M:%S UTC')}",
        f"Final CV: {state.ensemble_cv or state.cv_mean:.4f}",
        f"Metric: {state.metric_name}",
        f"Models: {', '.join(m['model_type'] for m in (state.model_configs or []))}",
        f"Ensemble: {state.ensemble_method}",
        '"""',
        "",
        "# === IMPORTS ===",
    ]
    notebook_lines.extend(sorted(all_imports))
    notebook_lines.extend([
        "",
        "# === CONFIGURATION ===",
        "RANDOM_SEED = 42",
        "import numpy as np",
        "np.random.seed(RANDOM_SEED)",
        "",
    ])
    
    # 4. Add each stage
    for agent in AGENT_ORDER:
        if agent not in grouped:
            continue
        
        stage_name = agent.upper().replace("_", " ")
        notebook_lines.append(f"# {'='*60}")
        notebook_lines.append(f"# {stage_name}")
        notebook_lines.append(f"# {'='*60}")
        notebook_lines.append("")
        
        for entry in grouped[agent]:
            # Add purpose comment
            notebook_lines.append(f"# --- {entry['purpose']} ---")
            
            # Clean the code
            cleaned = _clean_code_block(entry["code"])
            notebook_lines.append(cleaned)
            notebook_lines.append("")
    
    # 5. Add submission generation footer
    notebook_lines.extend([
        "# === SAVE SUBMISSION ===",
        "print(f'Submission saved: submission.csv')",
    ])
    
    # Write notebook
    notebook_path = os.path.join(session_dir, "solution_notebook.py")
    with open(notebook_path, "w") as f:
        f.write("\n".join(notebook_lines))
    
    return notebook_path
```

**Code cleaning function — strips Professor artifacts:**

```python
def _clean_code_block(code: str) -> str:
    """
    Clean a sandbox code block for standalone execution.
    
    Removes:
    - SANDBOX_PREAMBLE duplicates (import polars already in imports section)
    - emit_to_operator() calls
    - run_in_sandbox() wrappers
    - Professor-specific imports
    - Absolute sandbox file paths (replace with relative)
    - Diagnostic injection wrappers
    
    Preserves:
    - All ML logic
    - All data operations
    - All model training code
    - Random seed assignments
    """
    lines = code.split("\n")
    cleaned = []
    
    skip_patterns = [
        "from tools.",
        "from agents.",
        "from shields.",
        "from graph.",
        "emit_to_operator(",
        "run_in_sandbox(",
        "mark_rejected(",
        "__DIAGNOSTICS__",
        "# === USER CODE START ===",
        "# === USER CODE END ===",
    ]
    
    for line in lines:
        stripped = line.strip()
        
        # Skip Professor-specific lines
        if any(pat in stripped for pat in skip_patterns):
            continue
        
        # Skip duplicate imports (already in header)
        if stripped.startswith("import polars") or stripped.startswith("import numpy as np"):
            continue
        
        # Replace absolute sandbox paths with relative
        # e.g., "/tmp/sandbox/session123/clean_train.parquet" → "train.parquet"
        line = re.sub(r'["\'].*?/sandbox/.*?/([^/"\']+)["\']', r'"\1"', line)
        
        cleaned.append(line)
    
    return "\n".join(cleaned)
```

```python
def generate_writeup(state: ProfessorState, session_dir: str) -> str:

{_reasoning_for("data_engineer")}

### Feature Engineering
{_reasoning_for("feature_factory")}
```

### Reproduction Validation

The most important test: does `solution_notebook.py` produce the same `submission.csv` as the pipeline?

```python
def validate_reproduction(
    notebook_path: str,
    session_dir: str,
    expected_submission_path: str,
) -> dict:
    """
    Run solution_notebook.py in a CLEAN sandbox and compare output.
    
    Returns:
    {
        "reproduced": bool,
        "rows_match": bool,
        "values_match": bool,
        "max_diff": float or None,
        "error": str or None,
    }
    """
    # Read the notebook
    with open(notebook_path, "r") as f:
        notebook_code = f.read()
    
    # Execute in clean sandbox — no state, no memory, no APIs
    result = run_in_sandbox(
        code=notebook_code,
        timeout=600,  # 10 minutes
        agent_name="reproduction_validation",
        purpose="Verify solution_notebook.py reproduces submission.csv",
        working_dir=session_dir,  # Has train.csv, test.csv
    )
    
    if not result["success"]:
        return {
            "reproduced": False,
            "rows_match": False,
            "values_match": False,
            "max_diff": None,
            "error": result["stderr"][:500],
        }
    
    # Compare submissions
    reproduced_path = os.path.join(session_dir, "submission.csv")
    if not os.path.exists(reproduced_path):
        return {
            "reproduced": False,
            "error": "Notebook did not produce submission.csv",
        }
    
    original = pl.read_csv(expected_submission_path)
    reproduced = pl.read_csv(reproduced_path)
    
    rows_match = len(original) == len(reproduced)
    
    if rows_match:
        # Find prediction column (not 'id')
        pred_col = [c for c in original.columns if c.lower() != "id"][0]
        orig_vals = original[pred_col].to_numpy().astype(float)
        repr_vals = reproduced[pred_col].to_numpy().astype(float)
        
        max_diff = float(np.max(np.abs(orig_vals - repr_vals)))
        
        # Classification: exact match required
        # Regression: tolerance 1e-4
        if state.competition_type in ("binary", "multiclass"):
            values_match = np.array_equal(orig_vals, repr_vals)
        else:
            values_match = np.allclose(orig_vals, repr_vals, atol=1e-4)
    else:
        max_diff = None
        values_match = False
    
    return {
        "reproduced": rows_match and values_match,
        "rows_match": rows_match,
        "values_match": values_match,
        "max_diff": max_diff,
        "error": None,
    }
```

**Common reproduction failures and how the assembler prevents them:**
- Missing random seed → assembler checks for `random_state` in model constructors, adds `random_state=RANDOM_SEED` if missing
- Sandbox path reference → `_clean_code_block()` replaces absolute paths with relative
- Missing import from SANDBOX_PREAMBLE → import deduplication captures all imports from all blocks
- Non-deterministic ordering → topological sort ensures deterministic block order

## COMMIT 4: Submission Safety Net — Shield 7 (shields/submission_safety.py)

### The problem this solves

Three endgame disasters:
1. Both final submissions are correlated (0.998) → private LB shakeup kills both
2. EWMA monitor freezes submissions in final week due to noisy public LB → Professor stops competing

```python
def verify_submission(
    submission_path: str,
    sample_submission_path: str,
    canonical_test_rows: int,
    competition_type: str = "binary",
) -> dict:
    """
    Verify a submission file is valid before marking as final.
    
    Runs 7 checks. ALL must pass for the submission to be valid.
    
    Returns:
    {
        "valid": bool,
        "checks": {
            "row_count_ok": bool,
            "column_count_ok": bool,
            "column_names_ok": bool,
            "no_nans": bool,
            "range_ok": bool,
            "not_truncated": bool,
            "format_ok": bool,
        },
        "issues": [str],   # Human-readable descriptions of failures
    }
    """
    issues = []
    checks = {}
    
    # Load submission
    try:
        submission = pl.read_csv(submission_path)
    except Exception as e:
        return {"valid": False, "checks": {}, "issues": [f"Cannot read submission: {e}"]}
    
    # Load sample for reference
    try:
        sample = pl.read_csv(sample_submission_path)
    except Exception:
        sample = None  # Verify what we can without sample
    
    # Check 1: Row count
    checks["row_count_ok"] = len(submission) == canonical_test_rows
    if not checks["row_count_ok"]:
        issues.append(f"Row count {len(submission)} != expected {canonical_test_rows}")
    
    # Check 2: Column count
    if sample is not None:
        checks["column_count_ok"] = len(submission.columns) == len(sample.columns)
        if not checks["column_count_ok"]:
            issues.append(f"Column count {len(submission.columns)} != expected {len(sample.columns)}")
    else:
        checks["column_count_ok"] = True  # Can't verify without sample
    
    # Check 3: Column names
    if sample is not None:
        checks["column_names_ok"] = submission.columns == sample.columns
        if not checks["column_names_ok"]:
            issues.append(f"Columns {submission.columns} != expected {sample.columns}")
    else:
        checks["column_names_ok"] = True
    
    # Check 4: No NaN/null in prediction columns
    pred_cols = [c for c in submission.columns if c.lower() not in ("id", "index", "row_id")]
    nan_count = sum(submission[c].null_count() for c in pred_cols)
    checks["no_nans"] = nan_count == 0
    if not checks["no_nans"]:
        issues.append(f"{nan_count} NaN values found in prediction columns")
    
    # Check 5: Value range
    range_ok = True
    for col in pred_cols:
        vals = submission[col].to_numpy()
        if competition_type in ("binary",):
            if np.any(vals < 0) or np.any(vals > 1):
                range_ok = False
                issues.append(f"Column '{col}': predictions outside [0,1] — min={vals.min():.4f}, max={vals.max():.4f}")
        elif competition_type == "regression":
            if np.any(np.isinf(vals)):
                range_ok = False
                issues.append(f"Column '{col}': contains inf values")
    checks["range_ok"] = range_ok
    
    # Check 6: Not truncated (last row has all columns)
    try:
        last_row = submission.tail(1)
        checks["not_truncated"] = last_row.null_count().sum_horizontal()[0] == 0
        if not checks["not_truncated"]:
            issues.append("Last row has null values — file may be truncated")
    except Exception:
        checks["not_truncated"] = False
        issues.append("Cannot verify last row — file may be truncated")
    
    # Check 7: Format (CSV delimiter)
    with open(submission_path, "r") as f:
        first_line = f.readline()
    checks["format_ok"] = "," in first_line  # Basic CSV check
    if not checks["format_ok"]:
        issues.append("File does not appear to be comma-delimited CSV")
    
    valid = all(checks.values())
    return {"valid": valid, "checks": checks, "issues": issues}
```

```python
def check_submission_diversity(
    submission_1_path: str,
    submission_2_path: str,
) -> dict:
    """
    Check diversity between final 2 submissions.
    
    Returns:
    {
        "correlation": float,     # Pearson correlation of predictions
        "diversity_rating": str,  # "good" | "moderate" | "warning"
        "message": str,           # Human-readable explanation
    }
    """
    sub1 = pl.read_csv(submission_1_path)
    sub2 = pl.read_csv(submission_2_path)
    
    # Find prediction column
    pred_col = [c for c in sub1.columns if c.lower() not in ("id", "index", "row_id")][0]
    
    preds_1 = sub1[pred_col].to_numpy().astype(float)
    preds_2 = sub2[pred_col].to_numpy().astype(float)
    
    corr = float(np.corrcoef(preds_1, preds_2)[0, 1])
    
    if corr > 0.995:
        rating = "warning"
        msg = (f"Correlation {corr:.4f} — submissions are nearly identical. "
               f"If one fails on private LB, the other will too. "
               f"Consider replacing one with a more diverse submission.")
    elif corr > 0.95:
        rating = "moderate"
        msg = f"Correlation {corr:.4f} — moderate diversity. Consider if more diverse option available."
    else:
        rating = "good"
        msg = f"Correlation {corr:.4f} — strong diversity between submissions."
    
    return {"correlation": round(corr, 4), "diversity_rating": rating, "message": msg}
```

```python
def estimate_lb_noise(

    n_total_test_rows: int,

            "gap_threshold": 0.005,
```

```python
def select_final_submissions(
    submissions_history: list[dict],
) -> dict:
    """
    Select the 2 final submissions for the competition.
    
    Strategy:
    - Selection 1: highest public LB score (aggressive)
    - Selection 2: highest CV score from a DIFFERENT run than Selection 1 (diverse + defensive)
    - If only one run: Selection 2 = best CV with different post-processing
    
    Returns:
    {
        "selection_1": {"path": str, "source": str, "cv": float, "lb": float},
        "selection_2": {"path": str, "source": str, "cv": float, "lb": float},
        "correlation": float,
        "reasoning": str,
    }
    """
    if not submissions_history:
        return {"selection_1": None, "selection_2": None, "reasoning": "No submissions available."}
    
    # Sort by LB score for Selection 1
    by_lb = sorted(submissions_history, key=lambda s: s.get("lb_score", 0), reverse=True)
    selection_1 = by_lb[0]
    
    # For Selection 2: best CV from a DIFFERENT run
    other_runs = [s for s in submissions_history if s.get("run_id") != selection_1.get("run_id")]
    
    if other_runs:
        by_cv = sorted(other_runs, key=lambda s: s.get("cv_score", 0), reverse=True)
        selection_2 = by_cv[0]
        reasoning = (f"Selection 1: best LB ({selection_1.get('lb_score', 'N/A')}) from {selection_1.get('run_id')}. "
                     f"Selection 2: best CV ({selection_2.get('cv_score', 'N/A')}) from different run {selection_2.get('run_id')}.")
    else:
        # Only one run — pick best CV with different post-processing if available
        by_cv = sorted(submissions_history, key=lambda s: s.get("cv_score", 0), reverse=True)
        selection_2 = by_cv[0] if len(by_cv) > 1 else selection_1
        reasoning = "Only one run available. Both selections from same run."
    
    return {
        "selection_1": selection_1,
        "selection_2": selection_2,
        "reasoning": reasoning,
    }
```

## COMMIT 5: Memory Hygiene System — Shield 2 (shields/memory_hygiene.py)

### The problem this solves

After 15 competitions, ChromaDB has learned patterns. Some are wrong — correlation mistaken for causation. "Target encoding helps on healthcare data" from one lucky competition. Professor applies it on the next healthcare competition and it causes overfitting. Bad patterns compound. The system gets WORSE over time.

### Design — 5 mechanisms

This shield wraps every `chromadb.retrieve()` call. No pattern is applied blindly.

### Note: This activates meaningfully after 5+ competitions. Build it now so the infrastructure exists when patterns start accumulating.

```python
class MemoryHygieneGuard:
    """
    Wraps ChromaDB retrieval with validation, decay, contradiction detection, and quarantine.
    """
    
    def __init__(self, chromadb_collection, gate_config: dict):
        self.collection = chromadb_collection
        self.gate_config = gate_config
        self.failed_patterns_collection = "failed_patterns"
```

#### Mechanism 1: Retrieval Validation

```python
def retrieve_validated(self, query: str, n_results: int = 5) -> list[dict]:
    """
    Retrieve patterns from ChromaDB BUT mark each as HYPOTHESIS, not INSTRUCTION.
    
    Each returned pattern has:
    - The original pattern data
    - validated: False (must be confirmed via Wilcoxon before application)
    - validation_required: True
    """
    raw_results = self.collection.query(query_texts=[query], n_results=n_results)
    
    validated_results = []
    for result in raw_results:
        pattern = {
            **result,
            "validated": False,
            "validation_required": True,
            "source": "chromadb_memory",
            "retrieval_note": "HYPOTHESIS — must pass Wilcoxon gate before application. "
                              "Do NOT apply blindly.",
        }
        validated_results.append(pattern)
    
    return validated_results


def confirm_pattern(self, pattern_id: str, wilcoxon_passed: bool, cv_delta: float) -> None:
    """
    After Wilcoxon test on a retrieved pattern:
    - Passed: increment confidence by 0.10, mark validated=True
    - Failed: decrement confidence by 0.05
    - Caused degradation (cv_delta < 0 AND significant): quarantine
    """
    pattern = self._get_pattern(pattern_id)
    
    if wilcoxon_passed and cv_delta > 0:
        pattern["confidence"] = min(pattern.get("confidence", 0.5) + 0.10, 1.0)
        pattern["validated"] = True
        pattern["last_confirmed_competition"] = self._current_competition_id()
        self._update_pattern(pattern)
    
    elif cv_delta < 0:
        # Pattern caused degradation — quarantine it
        self.quarantine_pattern(pattern, 
            reason=f"Caused CV degradation of {cv_delta:.4f} on competition {self._current_competition_id()}")
    
    else:
        # Wilcoxon failed but no degradation — just decay confidence
        pattern["confidence"] = max(pattern.get("confidence", 0.5) - 0.05, 0.0)
        self._update_pattern(pattern)
```

#### Mechanism 2: Contradiction Detection

```python
def check_contradiction(self, new_rule_text: str, existing_rules: list[dict]) -> list[dict]:
    """
    Before storing a new rule, check for semantic conflicts with existing rules.
    
    Uses cosine similarity on text embeddings (ChromaDB provides this natively).
    If similarity > 0.85: potential conflict detected.
    Then use LLM to confirm: "Do these two rules contradict each other?"
    
    Returns list of conflicting rules (empty if no conflicts).
    """
    conflicts = []
    
    # Get embedding for new rule
    new_embedding = self.collection._embed([new_rule_text])[0]
    
    for existing in existing_rules:
        existing_embedding = self.collection._embed([existing["text"]])[0]
        
        # Cosine similarity
        similarity = np.dot(new_embedding, existing_embedding) / (
            np.linalg.norm(new_embedding) * np.linalg.norm(existing_embedding)
        )
        
        if similarity > 0.85:
            # High similarity — check if it's a contradiction or just related
            # Use LLM for confirmation
            confirmation = llm_call(
                f"Do these two rules CONTRADICT each other?\n"
                f"Rule A: {new_rule_text}\n"
                f"Rule B: {existing['text']}\n"
                f"Reply ONLY 'yes' or 'no'.",
                agent_name="memory_hygiene",
            )
            
            if "yes" in confirmation["text"].lower():
                conflicts.append({
                    "existing_rule": existing,
                    "similarity": round(float(similarity), 3),
                    "resolution": "Update existing rule with conditional logic or replace.",
                })
    
    return conflicts
```

When contradiction is found: DON'T store both rules. Update the existing rule with a CONDITION:

#### Mechanism 3: Confidence Decay

```python
def decay_unused_rules(self, current_competition_number: int) -> list[dict]:
    """
    Rules not confirmed in the last 10 competitions lose 0.1 confidence per competition.
    Prevents accumulation of stale rules from early competitions.
    """
    all_rules = self._get_all_active_rules()
    decayed = []
    
    for rule in all_rules:
        last_confirmed = rule.get("last_confirmed_competition", 0)
        competitions_since = current_competition_number - last_confirmed
        
        if competitions_since > 10:
            old_confidence = rule["confidence"]
            rule["confidence"] = max(rule["confidence"] - 0.1, 0.0)
            self._update_pattern(rule)
            decayed.append({
                "rule_id": rule["rule_id"],
                "text": rule["text"][:100],
                "old_confidence": old_confidence,
                "new_confidence": rule["confidence"],
                "competitions_since_confirmed": competitions_since,
            })
    
    return decayed
```

#### Mechanism 4: Quarantine

```python
def quarantine_pattern(self, pattern: dict, reason: str) -> None:
    """
    Move a pattern to the failed_patterns collection.
    
    Quarantined patterns are presented to agents as ANTI-PATTERNS:
    "DO NOT try target encoding on high-cardinality healthcare data —
    caused degradation on competition X."
    """
    pattern["quarantined"] = True
    pattern["quarantine_reason"] = reason
    pattern["quarantine_timestamp"] = datetime.utcnow().isoformat()
    
    # Add to failed_patterns collection
    self._add_to_collection(self.failed_patterns_collection, pattern)
    
    # Remove from active collection
    self._remove_from_collection(self.collection, pattern["rule_id"])


def get_anti_patterns(self, query: str, n_results: int = 3) -> list[dict]:
    """
    Retrieve quarantined patterns relevant to the current task.
    Agents receive these as: "AVOID: {pattern_text} — Reason: {quarantine_reason}"
    """
    results = self._query_collection(self.failed_patterns_collection, query, n_results)
    return [
        {
            "text": r["text"],
            "avoid_reason": r["quarantine_reason"],
            "original_source": r.get("source_competition", "unknown"),
        }
        for r in results
    ]
```

#### Mechanism 5: Cap at 20 Active Rules

```python
def enforce_rule_cap(self, max_rules: int = 20) -> list[dict]:
    """
    If active rules exceed max_rules, evict the lowest-confidence ones.
    """
    all_rules = sorted(
        self._get_all_active_rules(),
        key=lambda r: r.get("confidence", 0),
        reverse=True
    )
    
    evicted = []
    while len(all_rules) > max_rules:
        victim = all_rules.pop()  # Lowest confidence
        victim["status"] = "evicted"
        self._archive_pattern(victim)
        evicted.append(victim)
    
    return evicted
```
